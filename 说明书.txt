BAND 使用说明

一、安装
1. 解压压缩包。
2. 新电脑或提示缺少 WebView2 时，双击“安装BAND-完整.cmd”。
   英文备用入口是“Install-BAND-With-WebView2.cmd”。
3. 已经能正常打开 BAND 的电脑，双击“安装BAND-快速.cmd”。
   英文备用入口是“Install-BAND-Fast.cmd”，这个入口会跳过 WebView2 检测，安装更快。
4. 旧入口“安装BAND.cmd” / “Install-BAND.cmd”仍然是完整安装。
5. 安装完成后，桌面会出现“BAND”图标。
6. 以后直接双击桌面的“BAND”图标使用。

必须先把压缩包完整解压出来，再运行安装脚本；不要在压缩包预览窗口里直接双击安装。安装包里已附带 MicrosoftEdgeWebview2Setup.exe，如果另一台电脑缺少 Microsoft Edge WebView2 Runtime，安装脚本会先自动安装它，避免启动时出现“找不到指定文件”的异常。

说明：Windows 软件底层一定会有 .exe 主程序，但你日常不需要打开程序文件夹里的 exe。安装后直接使用桌面上的 BAND 图标即可。

二、不断开更新
如果另一台电脑正在使用 BAND，不想被安装器强制关闭：
1. 解压新版安装包。
2. 双击“更新BAND-不关闭.cmd”。
   如果打不开，请双击英文备用入口“Update-BAND-NoClose.cmd”。这个入口会跳过 WebView2 检测，不会等待运行库安装。
3. 安装器会把新版先准备到本机，不会关闭正在运行的 BAND。
4. 等你方便时，手动关闭 BAND。
5. 双击桌面生成的“完成BAND更新.cmd”或“Finish-BAND-Update.cmd”，完成替换。

账号登录资料不会被删除。登录资料保存在 %LOCALAPPDATA%\BandChat\data，不在安装包程序目录里。

三、登录账号
1. 打开 BAND。
2. 左侧点击账号名称选择当前目标，蓝色高亮行就是当前目标。
3. 点击“打开账号首页”。
4. 在右侧对应账号页面里登录 BAND。

账号左侧的小勾只用于启动时自动打开，不代表当前操作目标。

四、聊天室
每个账号页面左侧都有自己的聊天室列表，不会和其他账号混在一起。

常用按钮：
- 保存当前为聊天室：进入真实聊天室后保存。
- 读取当前页聊天室 / 读取到列表：读取当前页面里已经显示出来的聊天室链接，只保存到列表，不会自动打开。
- 打开所选 / 打开标记 / 打开全部：只作用于当前账号。
- 改名：修改聊天室标签名称，也可以选中聊天室后按 F2。
- 删除所选：删除保存的聊天室，也可以按 Delete。

聊天室列表里蓝色高亮行是当前目标；第一列标记只用于“打开标记”。

五、快捷键
- Ctrl+W：关闭当前网页标签。
- Ctrl+R 或 F5：刷新当前网页标签。
- Ctrl+C / Ctrl+V：网页内部复制粘贴。

六、注意
BAND 没有公开接口可以一次读取账号所有聊天室。软件会从当前页面里读取已经显示出来的真实聊天室链接。如果没有读全，请先进入 BAND 聊天列表并滚动让列表显示出来，再点击读取。

发送版压缩包不会包含你的登录资料。每台电脑第一次使用都需要自己登录账号。
