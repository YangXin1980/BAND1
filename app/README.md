# BAND

BAND 是一个 Windows 桌面软件，用来在一个软件窗口里管理多个 BAND 账号，并在每个账号页面里打开多个聊天室。

## 安装

1. 解压压缩包。
2. 新电脑或提示缺少 WebView2 时，双击 `安装BAND-完整.cmd`。英文备用入口是 `Install-BAND-With-WebView2.cmd`。
3. 已经能正常打开 BAND 的电脑，双击 `安装BAND-快速.cmd`。英文备用入口是 `Install-BAND-Fast.cmd`，这个入口会跳过 WebView2 检测，安装更快。
4. 旧入口 `安装BAND.cmd` / `Install-BAND.cmd` 仍然是完整安装。
5. 安装完成后，桌面会出现 `BAND` 图标。
6. 以后直接双击桌面的 `BAND` 图标使用。

必须先把压缩包完整解压出来，再运行安装脚本；不要在压缩包预览窗口里直接双击安装。安装包里已附带 `MicrosoftEdgeWebview2Setup.exe`，如果另一台电脑缺少 Microsoft Edge WebView2 Runtime，安装脚本会先自动安装它，避免启动时出现“找不到指定文件”的异常。

不要直接把压缩包里的程序文件当成日常入口使用；Windows 软件底层会有 `.exe` 主程序，但安装后你看到和打开的是桌面上的 `BAND` 软件图标。

## 不关闭软件更新

如果另一台电脑正在使用 BAND，不想被安装器强制关闭：

1. 解压新版安装包。
2. 双击 `更新BAND-不关闭.cmd`，英文备用入口是 `Update-BAND-NoClose.cmd`。这个入口会跳过 WebView2 检测，不会等待运行库安装。
3. 安装器会把新版先准备到本机，不会关闭正在运行的 BAND。
4. 等你方便时，手动关闭 BAND。
5. 双击桌面生成的 `完成BAND更新.cmd` 或 `Finish-BAND-Update.cmd`，完成替换。

账号登录资料不会被删除。登录资料保存在 `%LOCALAPPDATA%\BandChat\data`，不在安装包程序目录里。

## 使用

1. 打开 `BAND`。
2. 左侧点击账号名称选择当前目标，蓝色高亮行就是当前目标。
3. 点击顶部 `打开账号首页`。
4. 账号页面会显示在主窗口右侧，在对应账号页面里登录 BAND。

每个账号页面左侧是该账号自己的聊天室列表，右侧是该账号打开的网页标签。

账号左侧的小勾只用于启动时自动打开，不代表当前操作目标。

## 聊天室

- `保存当前为聊天室`：进入真实聊天室后保存到当前账号。
- `读取当前页聊天室` / `读取到列表`：从当前 BAND 页面里读取已经显示出来的聊天室链接，只保存到列表，不会自动打开。
- `打开所选`、`打开标记`、`打开全部`：只作用于当前账号的聊天室列表。
- `改名` 或选中聊天室后按 `F2`：修改聊天室标签名称。
- `删除所选` 或 `Delete`：删除保存的聊天室。

聊天室列表里蓝色高亮行是当前目标；第一列标记只用于 `打开标记`。

注意：BAND 没有公开本地接口可以一次读取账号所有聊天室。软件会从当前页面里读取已经显示出来的真实聊天室链接；如果没有读全，请先进入 BAND 聊天列表并滚动让列表显示出来，再点击读取。

## 快捷键

- `Ctrl+W`：关闭当前网页标签。
- `Ctrl+R` 或 `F5`：刷新当前网页标签。
- 网页内部可正常使用 `Ctrl+C`、`Ctrl+V` 和右键菜单。

## 数据位置

账号登录资料和配置保存在当前电脑本机：

```text
%LOCALAPPDATA%\BandChat\data
```

发送版压缩包不会包含你的登录数据。
